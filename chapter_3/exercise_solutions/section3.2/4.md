

### 1. Forward Direction ($\Rightarrow$)
**Goal:** Prove that if $f(n) = \Theta(g(n))$, then $f(n) = O(g(n))$ and $f(n) = \Omega(g(n))$.

**Assumption:** Let $f(n) = \Theta(g(n))$.
By the definition of $\Theta$, there exist positive constants $c_1, c_2, n_0$ such that:
$$0 \le c_1 g(n) \le f(n) \le c_2 g(n) \quad \text{for all } n \ge n_0$$

This inequality can be split into two parts:
1.  **Upper Bound:** $f(n) \le c_2 g(n)$ for all $n \ge n_0$.
    * This satisfies the definition of $O(g(n))$ with $c = c_2$ and threshold $n_0$.
2.  **Lower Bound:** $0 \le c_1 g(n) \le f(n)$ for all $n \ge n_0$.
    * This satisfies the definition of $\Omega(g(n))$ with $c = c_1$ and threshold $n_0$.

**Conclusion:** Therefore, $f(n) = O(g(n))$ and $f(n) = \Omega(g(n))$. $\checkmark$

### 2. Backward Direction ($\Leftarrow$)
**Goal:** Prove that if $f(n) = O(g(n))$ and $f(n) = \Omega(g(n))$, then $f(n) = \Theta(g(n))$.

**Assumptions:**
1.  Since $f(n) = O(g(n))$, there exist constants $c_2 > 0$ and $n_1$ such that:
    $$f(n) \le c_2 g(n) \quad \text{for all } n \ge n_1$$
2.  Since $f(n) = \Omega(g(n))$, there exist constants $c_1 > 0$ and $n_2$ such that:
    $$c_1 g(n) \le f(n) \quad \text{for all } n \ge n_2$$

**Construction:**
To satisfy the $\Theta$ definition, both inequalities must hold true simultaneously. We define a new threshold $n_0$:
$$n_0 = \max(n_1, n_2)$$

**Implication:**
For any $n \ge n_0$, it is true that $n \ge n_1$ **and** $n \ge n_2$.
Therefore, combining the inequalities:
$$c_1 g(n) \le f(n) \le c_2 g(n) \quad \text{for all } n \ge n_0$$

**Conclusion:**
This matches the definition of $\Theta(g(n))$.
Thus, $f(n) = \Theta(g(n))$.
